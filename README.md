# Chrome 收藏整理助手

这是一个可以直接加载到 Chrome 的小扩展，作用很简单：

- 现在点扩展图标，会直接在 Chrome 右边打开侧边栏
- 会显示你常用的网站，点一下就能打开
- 能按收藏本身最后一次打开的时间分析收藏夹，把很久没动过的收藏整理进“吃灰”文件夹
- 也能把历史记录里经常出现、但还没收藏的具体网址筛出来，再按域名去重后加入“翻牌”文件夹

## 怎么安装

1. 打开 Chrome。
2. 在地址栏输入 `chrome://extensions/`。
3. 打开右上角的“开发者模式”。
4. 点击“加载已解压的扩展程序”。
5. 选择这个文件夹：`/Users/Zhuanz/reps/chrome_tab`
6. 加载完成后，建议把这个扩展固定到工具栏，方便一键打开右侧边栏。

## 怎么用

1. 点击 Chrome 右上角扩展图标，打开“Tab Group Launcher”。
2. 现在点它后，会直接打开右边的侧边栏。
3. 最上面会显示常用网站，点图标就能打开。
4. 如果想整理收藏夹，就点“整理收藏夹”，先点“分析收藏夹”。
5. 看完建议列表后，点“一键移到‘吃灰’”。
6. 如果想补收藏，就点“翻牌高频网址”，候选网址会自动列出来。
7. 如果想换筛选范围，可以改“最近多久”和“至少出现多少次”，列表会自动刷新。
8. 看完候选网址后，勾选你想加入收藏的那些。
9. 再从下拉框选择要放进哪个收藏夹。
10. 最后点“加入已勾选收藏”。

## 注意

- 这个扩展保存的数据放在你本机浏览器里，不会上传出去
- “吃灰”文件夹会自动建在 Chrome 收藏夹里，方便你之后再翻出来
- “翻牌”文件夹也会自动创建，用来放那些常看但以前没收藏的网址
- “翻牌高频网址”会先过滤掉已经在收藏夹里的具体网址，再按域名去重，同一个网站下不同页面只会保留 1 条
- “翻牌”会自动跳过纯 IP 地址的网址，不会把这类地址加进收藏夹
- “翻牌高频网址”也会跳过本地测试地址，比如 `localhost:5231`、`127.0.0.1`
